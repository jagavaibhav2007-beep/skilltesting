# Typography Analysis: sohub.digital

## 1. Core Stack
*   **Primary Typeface**: `Plus Jakarta Sans` (Geometric Sans-Serif).
*   **Branding**: Main logo and primary nav links in lowercase.
*   **UI/CTAs**: Functional labels in uppercase with wide tracking.

## 2. Specification Table
| Element | Font Weight | Size (Est.) | Tracking | Leading |
| :--- | :--- | :--- | :--- | :--- |
| **Hero Headers** | 800 (Extra Bold) | 72px - 96px | -0.04em (Tight) | 1.0 (Tight) |
| **Sub-headings** | 700 (Bold) | 32px - 48px | -0.02em | 1.1 |
| **Body Paragraphs**| 400 (Regular) | 16px - 18px | Normal | 1.6 (Open) |
| **CTAs / Buttons** | 700 (Bold) | 14px - 16px | +0.05em (Wide) | N/A |

## 3. Visual Hierarchy Strategies
*   **Dual-Tone Headers**: Split headers into **Black (#000000)** and **Grey (#999999)** to create focus within a single phrase.
*   **High Contrast**: Pure black text on an off-white (#F2F2F2) background for maximum impact.
*   **Brutalist Compactness**: Tight letter-spacing and line-height on headings create a "monolithic" and high-authority visual block.
*   **Geometric Harmony**: The rounded terminals of Plus Jakarta Sans mirror the circular/pill-shaped UI elements for a cohesive feel.
